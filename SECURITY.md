# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.3.5   | Yes       |
| < 0.3.3 | No        |

## Reporting a vulnerability

Please **do not** open public GitHub issues for security problems.

Contact the Levushka team through your usual support channel (Telegram bot or personal account at [my.levushka.app](https://my.levushka.app)) with:

- Description of the issue
- Steps to reproduce
- Affected platform (macOS / Windows) and app version
- Impact assessment if known

We aim to acknowledge reports within 7 days.

## Scope

In scope:

- Levushka VPN Desktop client (connection, storage, updates, helper)
- Signed manifests loaded from `app.levushka.app`

Out of scope for this repository:

- VPN server infrastructure (report via support)
- Third-party dependencies (report upstream when appropriate)

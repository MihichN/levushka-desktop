**Levushka VPN** — нативное десктоп-приложение для сервиса [Levushka VPN](https://levushka.app). Подключается через полноценный **TUN-туннель** на уровне системы, а не только через прокси браузера. Поэтому работают Telegram, игры и другие программы, которые не используют системный прокси.

## Возможности

### Подключение и подписка

- Импорт подписки по ссылке `https://subs.levushka.app/sub/{token}/vless`
- Импорт по токену (32 hex-символа)
- Зашифрованные ссылки `levushka://crypt1/...`
- Deep link `levushka://import?url=...` — открытие подписки из браузера или бота
- Автообновление подписки (по умолчанию раз в час)
- Статус подписки: активна, истекла, лимит устройств
- Отображение трафика и срока действия из HTTP-заголовков (совместимо с Happ)

### VPN

- **Полный туннель TUN** — весь трафик системы через VPN (с настраиваемыми исключениями)
- Протокол **VLESS** через [Xray-core](https://github.com/XTLS/Xray-core)
- Список серверов с **TCP-пингом**
- Подключение / отключение в один клик
- Иконка в трее, таймер сессии, автоподключение при запуске

### Маршрутизация

Пресеты загружаются с сервера и обновляются без переустановки приложения:

- **Всё через VPN** — весь трафик в туннель
- **Россия напрямую** — российские IP и популярные сервисы без VPN
- **Своя конфигурация** — ручные списки доменов (напрямую или только через VPN)

### Интерфейс и настройки

- Русский и английский язык
- Светлая / тёмная / системная тема
- Масштаб текста
- Предпочтение IPv4 / IPv6
- Проверка обновлений приложения
- Запуск при входе в систему (macOS)

## Как начать

1. Скачайте и установите приложение для своей ОС.
2. Получите ссылку на подписку в [Telegram-боте](https://levushka.app) или в [личном кабинете](https://my.levushka.app).
3. Вставьте ссылку на экране импорта (или откройте `levushka://` ссылку).
4. Выберите сервер и нажмите **Подключить**.

При первом подключении macOS или Windows запросят права администратора — это нужно для установки сетевого helper и маршрутов TUN. Пароль запрашивается один раз.

## Архитектура (кратко)

```
┌─────────────────────┐
│  Levushka VPN (UI)  │  Tauri + React — пользовательский режим
└──────────┬──────────┘
           │ IPC
┌──────────▼──────────┐
│  Privileged helper  │  Маршруты, DNS, запуск xray
└──────────┬──────────┘
           │
┌──────────▼──────────┐
│  Xray-core (TUN)    │  VLESS → VPN-сервер
└─────────────────────┘
```

Подписка и биллинг — на стороне `subs.levushka.app` и личного кабинета. Приложение — тонкий клиент: загружает конфиги, поднимает туннель, показывает статус.

## Форматы импорта

| Формат | Пример |
|--------|--------|
| Полная ссылка | `https://subs.levushka.app/sub/{token}/vless` |
| Только токен | `a1b2c3d4...` (32 символа) |
| Зашифрованная ссылка | `levushka://crypt1/{payload}` |
| Deep link | `levushka://import?url=https://...` |

## Обновления

Приложение проверяет обновления на `app.levushka.app` и может установить новую версию без повторной загрузки с сайта. История версий — в [CHANGELOG.md](CHANGELOG.md).

## Безопасность

- Подписка хранится локально в защищённом каталоге приложения (не в открытом виде)
- Стабильный ID устройства (`x-hwid`) для лимита устройств на подписке
- Манифесты маршрутизации и geo-базы подписываются Ed25519
- Отчёты об уязвимостях: см. [SECURITY.md](SECURITY.md)
